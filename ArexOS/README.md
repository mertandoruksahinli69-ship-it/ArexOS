# ArexOS

**ArexOS** – A minimal, modular, and educational operating system simulation.  
Runs on Windows with a kernel written in C, supporting Python, Java, and EXE drivers.  

---

## Features

- Minimal kernel (written in C)
- Python, Java, and EXE driver support
- File management system (Python GUI)
- Basic GUI applications (Notepad, File Explorer, Webcam)
- Modular structure: separate kernel and driver files
- Educational and extendable system

---

## Installation

1. Download the `ArexOS` folder to your computer.  
2. Open Terminal (PowerShell) and navigate to the `system32` directory:

```powershell
cd "C:\Users\<username>\OneDrive\Desktop\ArexOS\system32"
